# Demos from Power Platform Community Sweden event 12/11 2025
This repo contains the following:

- **PPCS_MCP** - The PPCS MCP Server, the `stdio` version that was used for the Claude demo.
- **PPCS_MCP_Remote** - The streaming HTTP variant of the PPCS MCP Server, that was used in the Copilot Studio demo.
- **CustomCopilotStudioAgentUI** - The PPCS custom UI agent, that uses the Copilot Studio Client from the M365 Agents SDK.

Here is a [video](https://www.youtube.com/watch?v=GJLqc2VH9CA) that shows the demos from my presentation.

Thanks for coming the the event, I had great fun!
